## 후속 개발 단계
1. VS Code Debug Console 로그를 확인하여 현재 발생할 수 있는 모든 오류를 식별합니다.
2. 확인된 오류를 우선순위별로 해결합니다.
3. 최소 기능이 정상 작동하는지 확인한 후 점진적으로 개선 작업을 진행합니다.
4. 사용자 피드백을 받아 향후 개발 방향을 조정합니다.

## 참고 사항
- 모든 코드 변경 시 테스트를 실행하여 기존 기능이 손상되지 않도록 합니다.
- 오류 처리 및 로깅을 철저히 하여 디버깅이 용이하도록 합니다.
- 확장 프로그램의 성능과 안정성을 최우선으로 고려합니다.
