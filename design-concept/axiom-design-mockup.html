3. 타이포그래피

메시지 텍스트

기본 메시지 텍스트 스타일입니다. VS Code의 기본 폰트 패밀리와 크기를 상속받아 일관성 있는 사용자 경험을 제공합니다.

강조된 텍스트를 사용하여 중요한 정보를 강조할 수 있습니다.

작은 텍스트는 부가 정보에 사용됩니다.

코드 텍스트

코드 텍스트는 가독성이 높은 모노스페이스 폰트를 사용합니다. 이는 코드의 구조를 명확하게 보여주고 들여쓰기와 정렬을 유지합니다.

일반 텍스트 내에서 인라인 코드를 사용하여 코드 조각이나 명령어를 강조할 수 있습니다.

4. 반응형 디자인

넓은 패널 (600px 이상)
넓은 패널에서는 모든 UI 요소가 완전히 표시되며, 명령어 버튼 레이블과 설명이 모두 보입니다. 여유 공간을 활용하여 더 많은 정보를 제공합니다.
중간 패널 (400px-600px)
중간 패널에서는 기본 레이아웃을 유지하되, 일부 보조 UI 요소가 축소되어 표시됩니다. 명령어 버튼은 여전히 텍스트와 아이콘을 모두 보여줍니다.
좁은 패널 (400px 미만)
좁은 패널에서는 필수 기능만 표시하고 레이블 대신 아이콘을 사용합니다. 메시지 버블이 화면 너비에 맞게 조정되고, 명령어 버튼은 세로로 배치됩니다.
