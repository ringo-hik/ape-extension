# APE 파일 탐색기 구현 문서

## 개요

APE 확장 프로그램에 VS Code와 유사한 파일 탐색기 기능을 추가했습니다. 이 기능은 사용자가 워크스페이스 내 파일과 폴더를 쉽게 탐색하고 관리할 수 있게 해줍니다.

## 주요 기능

- **파일 시스템 탐색**: 워크스페이스 내의 모든 파일 및 폴더를 계층적으로 표시
- **파일/폴더 관리**: 새 파일 생성, 폴더 생성, 이름 변경, 삭제 기능 지원
- **컨텍스트 메뉴**: 파일 및 폴더에 대한 우클릭 메뉴 제공
- **파일 유형별 아이콘**: 다양한 파일 확장자에 맞는 아이콘 표시
- **트리 뷰 통합**: APE 사이드바에 통합된 트리 뷰 인터페이스

## 구현 세부 사항

### 1. 코어 클래스

- `ApeFileExplorerProvider`: 파일 시스템 데이터를 제공하는 클래스
- `FileItem`: 파일 시스템 항목 인터페이스
- `FileItemType`: 파일 시스템 항목 유형 열거형

### 2. 주요 메서드

- **파일 시스템 접근**:
  - `getChildren()`: 특정 폴더의 자식 항목 조회
  - `getTreeItem()`: TreeView에 표시할 항목 정보 제공
  - `_getFileItems()`: 특정 경로의 파일 항목 목록 가져오기

- **파일 작업**:
  - `createFileOrFolder()`: 파일 또는 폴더 생성
  - `deleteFileOrFolder()`: 파일 또는 폴더 삭제
  - `renameFileOrFolder()`: 파일 또는 폴더 이름 변경
  - `_removeDirectoryRecursive()`: 재귀적으로 디렉토리 삭제

- **UI 관련**:
  - `refresh()`: TreeView 새로고침
  - `_getIconPath()`: 파일/폴더 아이콘 경로 가져오기

### 3. 명령어 등록

- `ape.openFile`: 파일 열기
- `ape.createFile`: 새 파일 생성
- `ape.createFolder`: 새 폴더 생성
- `ape.deleteFileOrFolder`: 파일/폴더 삭제
- `ape.renameFileOrFolder`: 파일/폴더 이름 변경
- `ape.refreshFileExplorer`: 파일 탐색기 새로고침

### 4. 컨텍스트 메뉴

- 파일/폴더에 대한 우클릭 컨텍스트 메뉴
- 탐색기 상단 도구 모음 버튼 (새 파일, 새 폴더, 새로고침)

## 사용 방법

1. APE 사이드바에서 "파일 탐색기" 탭 선택
2. 워크스페이스의 파일 및 폴더 계층 구조 표시
3. 파일 클릭: 해당 파일을 편집기에서 열기
4. 우클릭 메뉴: 파일/폴더에 대한 작업 수행 (생성, 삭제, 이름 변경)
5. 상단 도구 모음: 새 파일, 새 폴더, 새로고침 작업 수행

## 향후 개선 사항

- **성능 최적화**: 대용량 폴더 처리 개선
- **드래그 앤 드롭**: 파일/폴더 드래그 앤 드롭 지원
- **검색 기능**: 파일 탐색기 내 검색 기능 추가
- **필터링**: 특정 패턴에 맞는 파일만 표시
- **Git 통합**: 파일의 Git 상태 표시
- **즐겨찾기**: 자주 사용하는 파일/폴더 즐겨찾기 기능
- **다중 선택**: 여러 파일 선택 후 일괄 작업 지원

## 기술적 부채

- VS Code Debug 콘솔 로그 확인 및 오류 해결 필요
- 대용량 폴더 처리 시 성능 최적화 필요
- 파일 상태 캐싱 메커니즘 구현으로 불필요한 파일 시스템 액세스 최소화
